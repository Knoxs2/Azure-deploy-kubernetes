# Azure-deploy-kubernetes
Example Project to create a Kubernetes cluster on Azure, inside a Virtual Machine, Create the necessary network to make the cluster acessible through the internet

Also creating a CI/CD pipeline in Azure Devops to deploy a simple web application, that will be  built inside a dockerfile, tested with unit tests, checked for code smells, code coverage, and then it's image will be deployed on a container registry

Using GitOps the Deployment will be automated via ArgoCD + Helm
